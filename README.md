# IV.BSFunnyPhones
Repository to distribute the mod Funny Phone Backgrounds for GTA IV.

# Included Backgrounds:

GTA VI - Sunrise Theme

Circoloco Records

Take-Two Interactive - Strauss' Theme

GTAForums Theme

# Installation Instructions:
1. Make sure you have Fusion Fix installed.
It's a requirement, besides, GTA IV without it plays poorly.
2. Extract the zip folder anywhere.
3. Copy the PhoneBGs folder to your GTAIV folder, wherever it may be.

# Observations:

This mod will replace existing phone backgrounds. I tried to replace the ones that I believe people use the least.

(Hooters Club, The CNT one, The bling one and the Rusty Donuts one)

If feedback is positive enough, I could create new ones! I'm open to suggestions.
